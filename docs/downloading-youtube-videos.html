Chapter 22 Downloading YouTube Videos

YouTube is one of the best sources for language data. In this unit, we will demonstrate how to automatically download YouTube video clips to create your own audio/video corpus for research/analysis.

Specifically, we may hope to download from YouTube :

  • The video of the clip
  • The audio of the clip
  • The subtitle of the clip (i.e., transcripts)

YouTube changes the structures of the website pretty often. This would make the module pytube function improperly very often as well. Using this module for YouTube processing may require frequent updates of the package. You may need to keep yourself updated of any recent changes from the pytube developers.

22.1 The Module pytube

In this unit, we will use the module pytube to download YouTube clips. You may need to install the package first if you haven’t.

pip install pytube
import pytube

22.3 Downloading Audios

  • The steps for audio are almost the same as video-downloading. The only difference is that we add a filter to a specific stream for audio only.
#import the package
from pytube import YouTube

url = 'https://www.youtube.com/watch?v=xsEJ6GeAGb0'
my_video = YouTube(url)

#set stream resolution
#my_video = my_video.streams.get_highest_resolution()
my_audio = my_video.streams.filter(only_audio=True)
my_audio[0].download(filename='YouTubeSample_audio')

22.4 Downloading Captions

  • A SRT file is a common file format for video captions (See this blog post for more information.).
  • With the YouTube object, we can access its captions (of a specific language) and generate an SRT format of the captions. Please note that not all YouTube clips come with the captions.

from pytube import YouTube

url = 'https://www.youtube.com/watch?v=xsEJ6GeAGb0'
my_video = YouTube(url)

# get all caption tracks
